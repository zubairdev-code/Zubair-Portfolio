## How to Add Your Photo

1. Place your photo in the `assets/` folder
2. Rename it to `profile.jpg` (or update the path in `index.html`)
3. The photo should be square (1:1 ratio) for best results
4. Recommended size: 500x500px or larger

## How to Deploy on GitHub Pages

### Method 1: Direct Upload

1. Go to your GitHub repository
2. Click on **Add file** > **Upload files**
3. Upload all files (index.html, styles.css, script.js, and assets folder)
4. Go to **Settings** > **Pages**
5. Under "Source", select "Deploy from a branch"
6. Select "main" branch and "/ (root)" folder
7. Click **Save**
8. Your site will be live at `https://zubairdev-code.github.io/`

### Method 2: Using Git

```bash
# Clone your repository
git clone https://github.com/zubairdev-code/zubairdev-code.github.io.git
cd zubairdev-code.github.io

# Copy all portfolio files here
# Then commit and push
git add .
git commit -m "Initial portfolio deployment"
git push origin main
```
